# PayRoll

## 项目简介
PayRoll 是一个用于管理员工计时和工资计算的系统。该系统提供了记录员工工作时间、计算工资、管理计时类型等功能。

## 功能
- 记录员工工作时间
- 管理计时类型
- 计算员工工资
- 按姓名搜索计时记录
- 添加、编辑和删除计时记录
